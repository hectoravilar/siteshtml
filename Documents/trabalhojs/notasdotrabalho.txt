Explicação dos Conceitos Aplicados
Variáveis: visitCount e users são variáveis para armazenar o número de visitas e a lista de usuários.

Operações Matemáticas e Operações Lógicas: A função isPrime() verifica se um número é primo usando operações matemáticas e lógicas.

Funções: updateVisitCount(), displayMessage(), addUser(), e isPrime() são usadas para modularizar a lógica.

Eventos: Um evento submit é adicionado ao formulário para captar os dados de entrada.

DOM: O JavaScript interage com o DOM para atualizar o contador de visitas e a mensagem de saudação.

Estrutura Condicional: Condicionais são usadas para verificar se o usuário é maior de idade e para validar o formulário.

Estrutura de Repetição: Um loop for verifica os números primos de 1 a 10 e exibe no console.

Tipos de dados: Utilizamos strings, numbers, arrays e objetos.

Arrays: users é um array que armazena os dados dos usuários.
